3. Software Description

ARRUS™ SDK (Software Development Kit) is a complete set of software needed for using and programming of the us4R-lite® platform. All available ARRUS™ SDK software revisions can be accessed and downloaded from the Github. Currently, the SDK is supporting MS Windows 10 and 11 x64 operating system or LINUX Ubuntu 20.04 or later. ARRUS™ SDK documentation (installation, API, examples, etc.) is available on-line, and thus, it is not a part of this user manual. Follow the link below to redirect to the ARRUS™ SDK documentation.

ARRUS™ SDK /Toolkit documentation >> https://us4useu.github.io/arrus-toolkit/
